
# NOUM Website (Energetic Galaxy)

This is a lightweight static website for NOUM token.
Files are ready for GitHub Pages or Vercel deploy.

## Included
- index.html
- css/style.css
- js/particles.js
- assets/logo.png (copied from uploads if present)
- assets/hero.jpg (cinematic banner)
- assets/svg32.svg (32x32 svg icon)
